# IV Fluid & Enteral Nutrition Calculator / 輸液・経腸栄養 組成計算シート

小児・新生児領域で使用する輸液および経腸栄養剤の組成・投与量を計算するWebアプリケーションです。  
複数ルート・複数チャネル（メイン＋側管）の輸液と経腸栄養を設定し、電解質濃度・GIR・カロリー・自由水量などをリアルタイムに計算・表示します。

## 主な機能

### 輸液（IV）
- **複数ルート管理** — CVルート、末梢ルートなど複数の輸液ラインを同時に管理
- **メイン＋側管構成** — 各ルートにメインラインと複数の側管を設定可能
- **側管の2モード** — 常時投与（continuous）と間欠投与（intermittent）を切り替え可能
- **リアルタイム計算** — 電解質濃度、ブドウ糖濃度、GIR、浸透圧、Ca×P積を自動計算
- **ルート内組成表示** — 通常時・間欠投与時それぞれの流量加重平均組成を表示
- **アラート表示** — K濃度、GIR、Ca×P積、浸透圧などの閾値超過を警告

### 経腸栄養
- **経腸ルート追加** — IVルートとは別に経腸栄養ルートを追加可能
- **mL/日 ⇔ 本/日 双方向同期** — 1日投与量と本数を相互変換して入力可能
- **自由水計算** — 経腸栄養剤の水分含有量（80〜85%）を反映した自由水量を算出
- **経腸向け組成表示** — カロリー密度、自由水比率、食物繊維などを表示（GIR・浸透圧は非表示）
- **カテゴリ別製剤選択** — 経腸ルートでは経腸栄養剤のみ、IVルートではIV製剤のみ表示

### 共通
- **合算結果表示** — IV＋経腸の全ルート合計（水分量、カロリー、電解質、栄養組成）を一覧表示
- **GIR（静脈のみ）** — 経腸ルート混在時はIV由来のグルコースのみでGIRを算出
- **IV/経腸内訳** — 水分量・カロリーはIV分と経腸分の内訳を表示
- **製剤設定** — デフォルト36製剤（IV 27 + 経腸 9）をカテゴリ別に管理、カスタム製剤の追加・編集が可能
- **検索付き製剤選択** — 輸液追加時にカテゴリ別表示＋テキスト検索で素早く選択
- **データ永続化** — localStorage による設定・状態の自動保存
- **設定込みHTML保存** — 現在の製剤設定をHTMLファイル自体に焼き込んでダウンロード。他のPCに配布すればそのまま同じ設定で使える

## 計算ロジック

### ルート内組成（通常時）
メイン＋常時側管の流量加重平均：

```
各成分濃度 = Σ(チャネル濃度 × 流量) / 合計流量
```

### ルート内組成（間欠時）
メイン（実効流量）＋常時側管＋全間欠側管の流量加重平均。  
側管の `mainRateDuring` が設定されている場合、間欠投与中のメイン流量が変更されます。

### GIR (Glucose Infusion Rate)
```
GIR (mg/kg/min) = IV由来ブドウ糖(g/day) × 1000 / 1440 / 体重(kg)
```
経腸栄養の糖質はGIR計算に含めません（静脈内持続投与ではないため）。

### 自由水計算
```
IV自由水 = IV投与量 × 1.0（100%）
経腸自由水 = 経腸投与量 × (water / 1000)
```
経腸栄養剤の水分含有量は製剤ごとに異なります（通常80〜85%）。

### Ca×P積
```
Ca×P = Ca(mEq/L) × P(mg/L) / 31
```
閾値: > 72 で警告表示。Ca・Pいずれかが0の場合は非表示。

### 浸透圧
```
mOsm/L = Na + K + Cl + Ca/2 + Mg/2 + HCO3 + Glu×5.56 + AA×10 + Fat×0.7
```

## 技術仕様

- **単一HTMLファイル** — CSS・JavaScript全てインライン、外部依存なし
- **サーバー不要** — ブラウザで直接開くだけで動作（`file://` プロトコル対応）
- **データ保存** — localStorage（ブラウザローカル、サーバー送信なし）
- **設定の持ち運び** — 「設定込みHTMLを保存」で現在の製剤設定をHTMLに埋め込んでダウンロード。ダウンロードしたファイルを他のPCで開けば同じ設定で使用可能
- **レスポンシブ対応** — PC・タブレット・スマートフォンで使用可能
